Alexa + New Relic

Have Alexa tell you current New Relic stats!

Download this project as a .zip file Download this project as a tar.gz file

Project Overview

Integrate NR metrics from NRQL query into a custom skill on the Alexa system (Amazon Echo).

α Goal:

Using the Alexa SDK + Amazon Lambda + a NR Insights query (via API) program the Alexa system to return NR metric about an application. Proof of Concept:

  • Get Alexa to recognize New Relic as a custom skill
  • Get Alexa to query NR Insights via the API key linked to NR Demo application
  • Get Alexa to speak the results of the query

Current Active Phrases:

"Alexa, Ask New Relic what is the response time of my application ?"

  • NRQL query of the NR Demo App via API
  • Returns average response time from 1 hour ago

"Alexa, Ask New Relic what is the revenue of my application?"

  • NRQL query of the NR Demo App via API
  • Returns total revenue

Next Steps:

  • Add more metrics that can be queried
  • Adjust number verbiage to natural language rounding (ie: 512,433 dollars = 512 thousand dollars)
  • Add aggregate queries into 'digest' format for a quick run-down of top metrics of a category
  • Add information based requests (non-NRQL query) that describe NR products
  • Figure out API key insertion so that this can be opened up to others to use via the Alexa Skills List
